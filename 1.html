웹 페이지 템플릿 제작 프로그램

<예시 화면>
1. 개요
  웹 페이지 템플릿 제작 프로그램은 홈페이지를 제작 할 때 고정된 템플릿을 제공하거나 직접 사용자가 템플릿을 제작하여 홈페이지를 구성할 수 있는 툴이다. 주로 쇼핑몰과 같이 상업적으로 이용되거나 이미지 중심의 페이지를 제작하는데 유용하다. 기존 중소형 쇼핑몰의 경우 매달 100개의 신상품을 제작하게 되는데, 상품 페이지 제작 시간에 매우 많은 시간을 소요하고 있다. 웹 페이지 템플릿 제작 프로그램은 쇼핑몰 신상품을 업데이트 하는데 들어가는 시간 소요를 1/10 수준으로 만드는 것을 목표로 하고 있다. 쇼핑몰이 지녀야 할 상업적 가치와 웹 페이지 제작에 소요되는 시간을 단축하여 소비자에게 더 많은 가치를 제공하고 매출 증대에도 기여한다.
  주요 기능으로 프로그램에 이미지 최적화 기능을 추가해 용량이 불필요하게 커지게 되는 점을 방지하고, 모바일 환경에서도 반응속도를 늘린다. 또한, 상품 업데이트 과정에서 사용자 중심의 UI를 제공해 오랜 시간이 걸리는 상품 상세 페이지 제작을 훨씬 쉽게 만든다.
  해당 프로그램을 통해 디자이너를 고용할 여력이 되지 못하는 영세 쇼핑몰의 인건비 절약에도 큰 도움을 줄 것이다. 포토샵과 같은 전문 툴을 다룰 수 있는 인력은 구하기 힘들 뿐 더러 연봉도 매우 높다. 대부분 직접 포토샵을 배우거나 값싼 디자이너 인력을 구하게 되는데, 이 경우 포토샵 교육을 위한 추가적 비용이 발생하게 된다. 우리 프로그램은 포토샵 사용을 최소하 하여 초보자들의 디자인 진입장벽을 낮추는 데도 큰 역할을 한다.
2. 담당 역할
  • 프로그램 UI/UX 설계
  • 프로그램 주요 기능 중 이미지 크기 최적화 기획 및 개발
  • 창업 캠프 최종 오디션 프로토 타입 제작
  • 사업 계획서 작성
3. 주요 기능
  • 템플릿 디자이너가 사전에 제작한 템플릿 제공
  • 사용자 정의 템플릿 사용 가능
  • 이미지의 특성을 판단하여 자동 완성하는 템플릿 사용
  • 각 구역 맞춤식 이미지 자동 조정
